# Python Small Projects

This repository contains a collection of simple Python scripts for various games and utilities. Each script is a standalone program that can be executed independently. Below is a brief description of each script and its purpose.

## Games

1. **2048.py**
   - A simple implementation of the 2048 game.

2. **Battleship.py**
   - A text-based version of the classic Battleship game.

3. **Conway's_Game_of_Life.py**
   - An implementation of Conway's Game of Life.

4. **Dice_Rolling_Simulator.py**
   - A script to simulate the rolling of dice.

5. **Hangman.py**
   - A text-based Hangman game.

6. **Memory_Match.py**
   - A memory match game.

7. **Minesweeper.py**
   - A text-based version of the Minesweeper game.

8. **Number_Guessing_Game.py**
   - A simple number guessing game.

9. **Number_Guessing_Game_with_Levels.py**
   - A number guessing game with difficulty levels.

10. **Rock_Paper_Scissors.py**
    - A simple Rock-Paper-Scissors game.

11. **Simple_Maze_Game.py**
    - A simple text-based maze game.

12. **Simple_Pong.py**
    - A simple implementation of the Pong game.

13. **Simple_Snake_Game.py**
    - A simple implementation of the Snake game.

14. **Simple_Text_Adventure.py**
    - A text-based adventure game.

15. **Sudoku_Solver.py**
    - A script that solves a given Sudoku puzzle.

16. **Text_based_Adventure_Game.py**

### How to Run

Each script can be run independently. For example, to run the `2048.py` script:

```sh
python 2048.py
```

### Requirements

Some scripts may require additional libraries. For example, the `Task_Manager` app requires Flask. Install the required libraries using `pip`:

```sh
pip install -r Task_Manager/requirements.txt
```
